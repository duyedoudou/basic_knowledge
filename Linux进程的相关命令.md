### Linux进程的相关命令:

| 命令                   | 功能                   |
| :--------------------- | ---------------------- |
| ps                     | 列出当前进程           |
| ps -aux                | 打印出进程的具体信息   |
| ps -u 用户名           | 查看此用户的进程       |
| ps -aus \| grep 关键字 | 查看和关键字有关的进程 |
| ps -aus  - -sort=-pcpu | 按照cpu占用率排序      |
| ps -aus  - -sort=-pmem | 按照内存占用率排序     |
|                        |                        |
| kill -9  加PID         | 无条件的停止该进程     |
|                        |                        |

死锁的4个必要条件：

互斥、请求保持、不可剥夺、环路等待
