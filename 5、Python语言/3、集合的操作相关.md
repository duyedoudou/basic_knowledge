## 集合的操作相关：

> 内容提要：
>
> 1、生成空集合；2、集合添加元素；

|            |                                           |
| ---------- | ----------------------------------------- |
| 删         | set.remove(某个元素)                      |
| 增         | set.add(key)、set.update(set2) `# 同字典` |
| 并、交、差 | set1\|set2、set1&set2、set1-set2          |



#### 生成空集合：

```python
a = set()  # 不能用{}，{}是用来生成空字典的
```

#### 集合的添加元素：

```
a.add('ws')
```
